    1. path : 실행명령어가 있는 폴더를 등록

      * 개요, 생성자, 상속, 패키지나 모듈
    * 오버로딩, 오버라이딩
    * 추상클래스, 인터페이스
    * 코틀린에서 지원하는 클래스들
    * 컬렉션
      * 배열, list, map, set
    * 예외처리
