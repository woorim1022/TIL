# 끝말잇기 프로젝트

## 조장(프로젝트 초기 세팅)

* 템플릿(로컬 git 저장소 생성)
  * README.md 생성
  * 커밋
  * Github 저장소를 push
* 단카방/구글드라이브...
  * github 저장소 초대

## 조원

* 원격 저장소 복제

  ```bash
  $ git clone _____________________
  ```

* 복제 후 주의 사항!

  * 명령어를 입력한 곳에 
  * 원격저장소 이름의 폴더가 생김.

  ```bash
  $ git clone https://github.com/cano721/end.git
  $ cd 원격저장소
  ```
