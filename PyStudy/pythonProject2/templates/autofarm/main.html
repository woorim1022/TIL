원격 제어 농장 프로그램

Smart 비닐하우스

AutoFarm은 원격 제어 기능으로 어디에서든 농장을 관리할 수 있게 해줍니다.

Iot의 기술을 활용해 센서에서 정보를 읽어 농장의 현상황을 파악할 수 있고 제어 기능을 이용해 원격으로 적절한 조치를 취할 수 있게 해줍니다.

농장 현황 모니터링

센서를 통해 얻은 데이터를 화면에 출력

온도

비닐하우스 실내 온도 표시

습도

비닐하우스 실내 습도 표시

이산화탄소 농도

농장 제어 기능

앱을 통해 농장의 원하는 기능을 원격으로 실행 가능

물, 빛

천장

온도

열선과 쿨러로 온도 조절

Smart 트랙터

AutoFarm의 원격 제어 기능을 트랙터에도 적용됩니다.

블루투스 통신을 이용해 원격으로 트랙터를 이동할 수 있습니다. 또한 IoT 기술을 활용해 적외선 센서, 무게 감지 센서를 이용해 편리한 서비스들을 제공합니다.
